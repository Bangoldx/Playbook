Node.js

Node.js allows your code to interact directly with the computer. Node comes with built in modules, basically libraries of code to help with day to day code use. Const is kind of like var but stands for constant and is just that, so unlike vars it can not be changed like a var can.

REPL or Read Evaluation Print Loop allows you to write code directly in the terminal with immediate results.

File system functionality, You can access the file system on a computer using node as mentioned previously. One of the main file system functions that you can do is copy files. Once you have your const fs set up with const fs = require("fs"); you can then use fs.copyFileSync("source", "destination");, and this will make a copy of the source file and place it within the destination file, creating one if the destination file does not exist. It also appears that this will overwrite, without warning, anything that is in the destination file, should it already exist.

NPM Package Manager and External Packages

From the terminal, use the command "npm init", this will INITitalize a new .json package within your project folder. This, I believe, will house all your dependancies. Then following the prompts, you will set up your package.

Node Package Manager or NPM is a great place to find different packages that house different bits of commonly used code, so it can save you time having to rewrite code. Inside the package you find and want to use, there will be instructions on how to install and run the package. When you install the external package, it will add itself to your .json file as a dependency.

npm nodemon will automatically monitor your server and restart it whenever there are new changes implemented. Install it using npm install -g nodemon, if an error occurs, runs as sudo.

Express.js with Node.js

Express is a Node framework and can allow you to create a server, among other web developments. Once you have your project file folder setup and have your .js file created, from the terminal you'll install it just like any other package manager, npm install express. This will create a package.json file within your project. Inside your .js file, create a const express = require("express"); the "" for "express" are extremely important and your server will error and not run if they are not there! and const app = express();.
