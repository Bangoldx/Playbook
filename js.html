Using the DOM

The DOM allows you to pick and chose parts of your HTML files and change it on the fly. It reads your HTML document as a tree structure and seperates each element as its own branch. At the top of the tree is the document, then the HTML element, then the head and the body. This will continue for as many different elements that you have inside your doc. You can select your tree's branches by using document.firstElementChild would select the HTML element, and you can continue this for any child within any element'. document.querySelector(" ") will search through your entire document looking for what is put within the quotations. then using the "dot notation ." and you give it a method. Objects within the DOM can have Properties and Methods properties describe the object and methods define things the element can do. Get Property car.color; would output the property color of the element. Set Property allows you to change the property itself. So car.numberOfDoors = 0; would then make it so that the car element has no doors.

You can also use getElementsByTagName("");. This will collect all elements with that tag and add them to an array. You can select an item out of that array with document.getElementsByTagName("li")[2], which will select the 3rd item in the array since arrays start at 0, then using the dot notation tell it what to do. So, document.getElementsByTagName("li")[2].style.color = "purple"; would change the third li to purple. getElementsByClassName("") and getElementById(""), the big difference between the tagName, className and the Id is that element is singular in the Id.
