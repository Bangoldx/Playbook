Formatting

To section off chunks of text or paragraphs use the pwhich will group everything in the opening and closing p and /ptags together.

To make things italicized, us em or emphasize instead of intalics i
Same goes for bold, use strong instead of b.

Images

You can add an image to your website by using an image tag img followed by src="" alt="".> Requiring a scr and alt, the alt will display if the img can not and also assist in Google's algorithm for search results. When using the img tag you can either use a direct url to a image found online or you can save an image locally and name it as source. If you have a image folder withing your project you will have to use a direct path such as "images/abcdefg.png"

Hyperlinks

Hyperlinks are made using anchor tags a. My Boardgame Collection The link is placed after the attribute a href="https://boardgamegeek.com/user/Bangold"> The text here is what shows up instead of the link intself. Anchor tags must be closed.

Forms

Forms can be made with the < form> tag. Forms can be outfitted with a multitude of attributes including but limited to: Text, calendars, color pickers, submit buttons.






Using < input type="text"> will create a place for the user to input text.
Using < label> will allow you to label your text input boxes effectively.


You can make a form perform an action such as Mail To using < form action=mailto:bangold@proton.me method="post" enctype="text/plain">
