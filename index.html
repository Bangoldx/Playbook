Command Line Techniques

  • List Items - ls
  • Change Directory - cd
  • Make Directory - mkdir
  • Create File - touch
  • Open File - open
  • Open File in Specific Application - open -a
  • Delete Files - rm
  • Delete All Files - rm *
  • Remove Directory - rm -r
  • Print Working Directory - pwd
Alta+click moves your cursor to where you clicked within the command line. CTR+A takes cursor to beginning of command line, CTR+E takes cursor to the end. CTR+U clears currently written command.
