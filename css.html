CSS

Cascading Style Sheets ~ You can change the style of your webpage by using CSS.
Backgrounds, can be changed using the style tag,
style="background-color:red;"
If you want something to be styled over the entire project, you can create a style and /style> entry within the head below the title portion of your HMTL code. Within the style block of code you can designate what elements of the code you want to adjust with out have to go through each individual instance of that element. Such as body, hr, etc. Each element you want to style has to be coded seperate but will effect all instances within the file. This is called internal CSS and They are written as such:

                              body { background-color:red;
                              }
                
                              hr { border-style: none;
                              background-color: red;
                              height: 2px;
                              }


External CSS Allows you to bring in a stylesheet from a .CSS file into your HMTL file. Use the link tag to link your css to your html. Do this by link rel="stylesheet" href="*file location*". In the external CSS file you can create css stylings for any element you have through your entire HTML project. Just make sure you place the < link> tag in your < head> under your < title>.

CSS Priority

When using CSS, it is possible to bug your code by having CSS code in multiple place. CSS code priority is:

  1. Inline CSS-CSS code within the element itself.
  2. Internal CSS-CSS code written for an element inside a < style> tag inside the < head>
  3. External CSS-Your seperate CSS style file. Linked to your HTML files using < link rel="stylesheet" href="filelocation > under the < title > portion of the HTML < head >"

In external CSS files, more specific instructions get priority over less specific instructions. Classes and IDs will override standard tags in styling giving you more options to style your site.

As a best practice when coding with CSS is it is best to do all your CSS styling in an external .CSS file.

CSS Syntax

selector(who){property(what):value(how);}

The selector is going to be your element or class. Your property is what you're changing, such as your color, background, borders and more. Your Value, or how, is going to determine in what way you are changing the property of your selector. CSS Values must be opened with ":" and closed with ";"

Class Selectors

Classes help making individual parts of the code styled. Adding (class="x") after your element name effectively create a sub element. You access them in CSS by adding a full stop "."before the class name. This allows you to modify just the class instead of the entire element. You're going to use this tag when grouping related items. You can have multiple classes for a particular elemement. i.e. < img class="broccoli circular"> will create the style for the classes "broccoli" as well as the class "circular".

ID Selectors

IDs are created within the opening tag with " id="heading". These are called in your CSS file using the "#" symbol.

Use to identify where there is only a single instance and use it when you want to create a specific style for a single item. Any given ID can only be used once, the system will not allow a second instance of this ID, you will have to create a different ID in order to continue. You can not have more that one ID for a single element. IDs are more selectively used in the wild.

Pseudo Selectors

Pseudo selectors have a ":", CSS elements can have different states, ie img:hover {background-color:gold;} would cause the background of an image that the mouse is hovering over to change color.

CSS Display Properties

  • Block
  • Inline
  • Inline-Block
  • None

You can change the display property of any given element using the "display: (property)"

Block elements take up the whole space on the screen, preventing any other elements from sitting next to it. We have seen block elements already, such as: Paragraphs (< p>), Headers (< h1 through h6>), Divisions (< div>), list and list items (< ol>, < ul>, and < li>), and Forms (< form>).

Inline (display:inline;) elements are those that can be placed side by side on the same line. Like < span> (opened and closed) which allows you to create a box within a box for specific styling. < Span>s can be nested withing other elements or created seperate.

Inline-Block (display:inline-block;) allows you to still have multiple blocks inline but be able to adujst the width of the elememnt.

None allows you to basically hide something. It will remove its visibility from the screen as well remove its width. Effectively deleting it. You may also use the visibility: hidden; CSS style to remove it from the screen but keep its box, which may be useful depending on what you're trying to do.

CSS Positioning

  1. Content is everything
  2. Order comes from code
  3. Children sit on top of parents
You can also change the position properties, Static is the default position proprty. Relative allows you to change the position relative to where the default position used to be. Cordinate properties T,B,L,R,. This does not change positions of any other elements on the screen. Absolute allows you to adjust position base on the "parent" element such as the < div>. Fixed allows you to position something in a permanent position that will allow you to scroll and keep the fixed element where it is, depsite all the scrolling.

Text Allignment

You can use < text-align:"";> to align you text. < text-align:center;> will center everything within the parent element as long as the elements that it is affecting has a full body width. If you do have a block element that has a specified width, you can have that align how you want it using margin:auto; or any variation of circle shorthand that you would need.

Font-Sizing

You can change the font sizing but using the font-size:""; code inside you CSS. Now, you can adjust this as a Static element by specifying a pixel amount, you can also adjust this as a Dynamic element. You do this by specifying a percentage, or an em, which is the phonetic pronounciation of the letter "M". So basically em is the height of a capital M, which is 16 pixels. So 2 em = 32px 5.625em = 90px. rem or "root"em allows you adjust the font property and ignore any inherited.

Float and Clear

float used with an image will allow any text to wrap around the image instead of being blocked off to a particular side. In CSS use float: "direction"; in which the direction is which side you want the image to be on. Clear allows you to clear a side of whatever element you use it with to "clear" that side of anything text.
